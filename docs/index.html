Project Description

To empower the client to automate certain warehousing processing activities, reduce errors and improve operational efficiencies through the use of image recognition technologies and robust application design.

THE PROJECT
THE SOLUTION
THE PROCESS

The Problem

Description

Ware house staff members lack knowledge on identifying and sorting stock items. This leads to an error in identification and sorting of stock items.This is time consuming and costly when a wrong item gets shipped to other warehouses/consumers and has to be returned.

The Solution

Description

A mobile and web appilication was developed to solve the problem.This application gives the user the ability to use a mobile phone or a computer to capture or select an image of interest(warehouse engineering items) and submit it to the image classification model inorder to predict the what the image is. The user is also capable of doing basic stock counting of weighed items and the Dashboard API which extends the solution by making the user to able to create their own image classification model to identify items of their own desired catageory.

User Story

1

Bob is part of the warehouse staff.Bob does not know every item in the warehouse off by heart.

2

Bob wants to identify what the item is.

3

Bob uses the image recognition Ninshiki app on his phone to identifying/classify the item.

4

The app uses artificial intelligence to predict item.

5

Bob gets the items classification and name as a result from app.Problem solved!
